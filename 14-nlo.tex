\section{Next-to-leading order}
\label{sec:nlo}

At LO accuracy, DIS is trivial \fherror{add LO diagram} and we find
\begin{align}
    C_u^{(0)}(z) &= e_u^2 \delta(1-z) = C_{\bar u}^{(0)}(z) = C_{c}^{(0)}(z) = C_{\bar c}^{(0)}(z) = \cdots\\
    C_d^{(0)}(z) &= e_d^2 \delta(1-z) = C_{\bar d}^{(0)}(z) = C_{s}^{(0)}(z) = C_{\bar s}^{(0)}(z) = \cdots
\end{align}
which, eventually, yields \cref{eq:LOxs}.

Starting from next-to-leading order (NLO), the diagrams become more complicated and we encounter new features.



\begin{itemize}
    \item NLOq
    \item NLOg
\end{itemize}

We need the Riemann Zeta function $\zeta(n)$ below, which is defined at integer values $n$ by
\begin{equation}
    \zeta_n = \zeta(n) = \sum\limits_{j=1}^\infty \frac 1 {j^n}
\end{equation}
